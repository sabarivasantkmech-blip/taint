Scientific references
CEACentral Electricity Authority. CO₂ Baseline Database for the Indian Power Sector. Used for state/grid electricity intensity. Source
ICCTInternational Council on Clean Transportation. Life-cycle greenhouse gas emissions of combustion engine and electric passenger cars and two-wheelers in India. Used for vehicle technology comparisons. Source
IPCCIPCC AR6 WGIII, Chapter 10: Transport. Used for transport decarbonisation framing and mitigation benchmarks. Source
IPCCIPCC 2006 Guidelines for National Greenhouse Gas Inventories. Used for fuel combustion accounting principles. Source
MoRTHMinistry of Road Transport & Highways. Vehicular emission norms and PUC context for India. Source
PUCNational Government Services Portal. PUC certificates confirm whether vehicle emissions meet required norms; TAINT uses the PUC value as a maintenance correction, not as a direct CO₂ measurement. Source

Gap fixed: PUC is treated as a maintenance and tailpipe-pollutant proxy. It is not used as a direct CO₂ instrument.
Remaining approximation: city fleet mix, live congestion emissions, real AQI chemistry, and satellite-smog visuals are modeled estimates. Admin factors should be recalibrated when verified city fleet or remote-sensing data is available.
How to host this page & enable global stats
1
Test locally over HTTP, not file://, so Supabase Auth redirects and browser storage behave like production.
python -m http.server 8787
http://127.0.0.1:8787/index.html
2
Deploy the static site from this repository:
GitHub Pages: Settings → Pages → Deploy from branch → main / root
Netlify: drag & drop this folder or connect the GitHub repo
Vercel: import the GitHub repo as a static project
3
Create a free Supabase project for global tracking:
supabase.com → New project → SQL Editor → run setup SQL → paste URL + anon/publishable key
4
Paste your Project URL and anon/publishable key into supabase-config.js.
5
Run taint_supabase_schema.sql, supabase_business_storage_migration.sql, supabase_sensitive_data_encryption.sql, and both advisor follow-up SQL files in Supabase SQL Editor.
6
In Supabase Authentication → URL Configuration, set the Site URL and Redirect URLs to your hosted domain. Enable Google, GitHub, or enterprise SSO in Authentication → Providers only when those buttons should appear.
