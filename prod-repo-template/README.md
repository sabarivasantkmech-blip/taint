# TAINT Production

This repository is the isolated production deployment target for TAINT.

Production code should arrive through release PRs generated by the dev repository workflow `Promote Dev Bundle To Prod Repo`. Do not develop directly in this repository.

## Production Deploy

The GitHub Pages workflow validates the static bundle and deploys only after the `production` GitHub Environment is approved.

Required repository settings:

- Protect `main`.
- Require pull requests before merge.
- Require `Validate production static bundle`.
- Create a `production` Environment with required reviewers.
- Restrict `production` deployments to `main`.
